Food Stream
================
* Node.js 0.4.12
    * express 2.5.5
    * ntwitter 0.2.9
    * socket.io 0.8.7
* jQuery
* jQuery Masonry

Clone
-----

    % git clone git://github.com/dexit/foodstream.git
    % cd foodstream


Install Dependencies
--------------------

    % npm install socket.io ntwitter express
